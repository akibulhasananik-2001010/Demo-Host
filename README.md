# Demo Host

A simple static website hosted by using GitHub Pages.

## Overview

This project was created as a beginner-friendly demonstration of how to host a website directly from a GitHub repository using GitHub Pages.

## Features

* Simple and lightweight static website
* Hosted using GitHub Pages

## Technologies Used

* HTML5
* GitHub Pages
* Git

## Project Structure

```text
Demo-Host/
│
├── index.html      # Main webpage containing the website content
└── README.md       # Project documentation and setup instructions
```
