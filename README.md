A simple static website hosted using GitHub Pages.

## Features

* Simple and lightweight static website
* Hosted using GitHub Pages
* Beginner-friendly project structure
* Easy to customize and expand

## Technologies Used

* HTML5
* GitHub Pages

## Project Structure

```text
Demo-Host/
│
├── index.html
└── README.md
```

## Live Demo

After enabling GitHub Pages, the project will be available at:

```text
https://your-username.github.io/Demo-Host/
```

Replace `your-username` with your GitHub username.

## How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/your-username/Demo-Host.git
```

2. Open the project folder.

3. Run the `index.html` file in any web browser.
