# Demo Host

A simple static website hosted using GitHub Pages.

## Overview

This project was created as a beginner-friendly demonstration of how to host a website directly from a GitHub repository using GitHub Pages.

The website contains a basic HTML page introducing the author and serves as an introduction to static website hosting, version control, and GitHub Pages deployment.

## Features

* Simple and lightweight static website
* Hosted using GitHub Pages
* Beginner-friendly project structure
* Demonstrates basic HTML webpage creation
* Easy to customize and expand

## Technologies Used

* HTML5
* GitHub Pages
* Git

## Project Structure

```text
Demo-Host/
│
├── index.html      # Main webpage containing the website content
└── README.md       # Project documentation and setup instructions
```

## Live Demo

After enabling GitHub Pages, the project will be available at:

```text
https://your-username.github.io/Demo-Host/
```

Replace `your-username` with your GitHub username.

## How to Run Locally

1. Clone the repository.
2. Open the project folder.
3. Open the `index.html` file in any web browser.

## Learning Purpose

This repository was created for learning purposes to understand:

* GitHub repositories
* Version control with Git
* Static website hosting
* GitHub Pages deployment
* Basic web development workflow
